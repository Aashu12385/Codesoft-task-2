# Codesoft-task-2

This is a simple Rock Paper Scissors game developed using Python and the Tkinter library for the graphical user interface (GUI). The game allows the user to play against the computer by selecting Rock, Paper, or Scissors. The computer randomly chooses its move, and the program compares both choices to determine the winner. The game features an easy-to-use interface, displays the result of each round, and helps beginners understand Python concepts such as Tkinter GUI development, event handling, conditional statements, and random number generation.
